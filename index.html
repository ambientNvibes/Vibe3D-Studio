Vibe 3D Studio V21.9.14 Controls

• Top bar holds project and preset commands
• Bottom bar stays focused on transform workflow only
• Tap outliner = solo select, hold outliner = multi-select
• Tap to select, then drag gizmo to transform
• Double-tap canvas = focus selection
• View menu: home, front, right, top, grid
• Save a selection into Kit A/B/C, then spawn it later
• Use Multi-Select in Hierarchy to tap several outliner items
• Long-press outliner still adds items without Multi-Select mode
• Gold = primary selection, blue = extra selected items
• Export hides helpers and skips sockets for cleaner files
• Asset kits now include restored static controls plus preview cards
• Two-finger tap = Undo
• Three-finger tap = Redo
• Autosave now stores recovery snapshots locally; restore is manual from the File menu to protect the default startup scene
• Animation bar adds timeline scrubbing, play/pause, looping, key diamonds, separate position / rotation / scale channels, and draggable timeline keys, a compact dope sheet, timeline box selection, copy/paste for selected keys, easing controls, onion-skin motion previews, a compact animation mode, tabbed model/animate/render workspaces, dedicated render tools, clickable workspace tabs, and model import support
• Add > Ref Front / Ref Side / Ref Top imports Blender-style aligned reference image planes
• Reference images now support opacity control and optional match-view-only filtering
• File menu now supports importing GLB, GLTF, and OBJ models directly into the scene
